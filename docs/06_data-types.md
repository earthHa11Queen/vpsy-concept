# 06. データ型定義

## 概要

VPSYで扱う主要なデータ型を定義します。  
これらは感情シミュレーションの入力パラメータとして機能します。

> **注記**  
> 本ドキュメントに含まれるデータ型（状態データタイプ等）は、  
> 社会科学・精神医学的シミュレーション対象の定義です。  
> 当事者への直接的な関与・診断・治療を目的とするものではありません。

---

## A-1. 環境データタイプ（Environmental Data Types）

個人の人格形成や感情のバイアスに、長期的・継続的な影響を与える外部要因。

**VPSYにおける位置づけ**

環境データは「初期値かつ固定的なバイアス」として機能します。  
これらは克服プロセスが発生しない限り、感情処理の全段階に作用し続けます。

また、VPSYでは環境データにおける「社会的問題の属性を持つこと」を  
ノーマル値として扱います。マジョリティ属性を平常値としないこのデータマネジメントは、  
傾向性というモジュール処理として有限かつパターン化が可能であるという考え方に基づきます。

**データタイプ一覧**

| カテゴリ | タイプ |
|---------|-------|
| 家庭環境 | 虐待、ネグレクト、片親家庭、離婚、不貞行為、死別 |
| 社会環境 | 不登校、孤立、孤独、いじめ、低所得、施設育ち、施設生まれ |
| 法・犯罪 | 犯罪行為、脱獄、犯罪歴、犯罪被害 |
| 社会的属性 | 人種差別、成金家庭、名門家庭、信仰 |
| 精神的体験 | 自殺未遂 |

---

## A-2. 関係性データタイプ（Relational Data Types）

シミュレーション対象の個人と他者との関係性を定義します。

**VPSYにおける位置づけ**

コミュニケーションの際のデータパケット構造（どの相手との通信か）を  
定義するために使用します。IPアドレスのホスト部に近い役割を持ちます。

**データタイプ一覧**

| カテゴリ | タイプ |
|---------|-------|
| 職場関係 | 上司、同期、同僚、先輩、後輩、部下 |
| 学校関係 | 同級生、同い年 |
| 一般的関係 | 尊敬する人、年上、年下、飲み仲間、知り合い |
| 家族・親族 | 親、兄弟姉妹、親戚、祖父母、曽祖父母、孫、息子娘 |
| 特殊関係 | 不倫関係、恋人関係、二股・複数股関係、遊びの関係、不良仲間 |

**サブタイプ・共通指標（検討中）**

- `仲の良さ`：共通指標として全関係性に付加するスコア値
- `腹違い`：関係性のサブタイプ・オプション属性として位置づけ

---

## A-3. 状態データタイプ（State Data Types）

特定の時点における個人の一時的な心理的・身体的状態。

**VPSYにおける位置づけ**

L1（情報層）への入力バイアスとして機能する一時的なフラグです。  
気分・気持ちとは異なり、外部から直接設定される即時的なパラメータです。

**データタイプ一覧**

| 分類 | タイプ |
|-----|-------|
| 精神状態 | 自殺願望、希死念慮、鬱、躁 |
| 覚醒状態 | 興奮、冷静、ニュートラル |
| 身体状態 | 酔いが回っている、過労 |

---

## A-4. 感情定義（参考：ブルチックの感情の輪）

感情のマッピング先として、ブルチック（Plutchik）の感情の輪を参照します。

- 参考：[ブルチックの感情の輪](https://mentalcare-lab.com/plutchik/)

VPSYではブルチックの輪で表現できる感情に加えて、  
ブルチックでは表現できない感情（外れ値的な状態など）も演算対象とすることを目指します。

---

## A-5. 行動・行為データタイプ（検討中）

現時点では詳細未定義。  
感情処理へのトリガーとなる行動・行為のデータタイプを今後定義します。

**基本方針（検討中）**

- `何もしていない` → `何もしない` という行動の一種として扱う
- `寝る` → 無知覚状態として扱い、起床後は別状態として処理する
- 行動の返却値として外部入力が発生するモデルを採用する
- 入力値はバリエーションがあっても、値は定数で固定する方針

---

## A-6. 情動計算式（基本式定義）

各要素の関係性を数式として表現した基本定義です。

```
感情   = 傾向性 + 情報の量とタイプ
気分   = 感情の連続 + 状況
傾向性 = 過去 + 気持ちによる変化
過去   = 気持ちの連続の圧縮値
観念   = 情報の抽象化モジュール
理解   = 観念を具体化するモジュール
```
