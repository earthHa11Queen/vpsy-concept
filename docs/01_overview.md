# 01. 概念全体の地図

## VPSYとは

**VPSY（Virtual Simulation System for Psychodynamics）** は、  
人間の感情プロセスを計算可能な構造として定義し、模倣・観察することを目的とした  
設計構想システムです。

感情の **完全な再現** を目指すものではありません。  
「感情と同じ方向を志向しつつ、姿形は異なる」という意味での **模倣（Simulation）** と、  
その動きを外部から観察・分析することを目的とします。

---

## システムの目的と位置づけ

VPSYは **人間が人間のために使う半有機的な道具** として位置づけます。

- AIによる人間の代替ではない
- 感情の正解を出力するシステムではない
- 「誰かを理解する一助となる材料の一欠片」として機能する観察ツールである

この立場により、無機物（AI・プログラム）と有機物（人間）の間にある倫理的・  
哲学的な一線を保持します。

---

## 全体構造の概観

VPSYは以下の3つの概念軸で構成されます。

### 軸1：時間階層モデル（感情 → 気分 → 気持ち）

感情・気分・気持ちを **時刻T（持続時間）** によって区別します。  
これにより、曖昧な感情という概念を時系列データとして計算可能にします。

```
感情  ←── 短命（秒単位）
気分  ←── 中期（時間〜日単位）
気持ち ←── 長期（週〜月以上）
```

### 軸2：多層処理モデル（OSI参照モデル対応）

外部刺激が感情として生成されるまでのプロセスを7層として定義します。  
詳細は [03_layer-model.md](03_layer-model.md) を参照。

### 軸3：フィードバックループ（記憶・環境）

記憶と環境は静的なデータではなく、システム全体に継続的に作用する動的要素です。  
記憶は非可逆圧縮（ハフマン符号化＋エビングハウス忘却曲線応用）で管理され、  
環境は情報処理の全段階に固定的なバイアスとして機能します。

---

## システム境界と観測モデル

VPSYでは **演算と観測を分離した独立システム** として構築します。

```
[外部刺激入力]
      ↓
[感情演算システム]  ←─── [記憶・環境バイアス]
      ↓
[観測システム]（外部から自由に観測点を設定可能）
```

無限ループを避けるため、観測は演算から切り離します。  
観測点を外部から自由に設定できることが、このシステムの観察ツールとしての  
本質的な機能です。
